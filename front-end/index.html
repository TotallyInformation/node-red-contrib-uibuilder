UIbuilder for Node-RED

This is the fallback master page for uibuilder. If you are seeing it, you have an error in your configuration.

If you have just deleted dist/index.html from your local uibuilder folder, please restart Node-RED so that the src folder is used.

If you don't have a local src/index.html, please add one. You can obtain the latest default master template from github.com/TotallyInformation/node-red-contrib-uibuilder/nodes/templates/index.html

Please see the README for the node-red-contrib-uibuilder node for details on how to use UIbuilder.
