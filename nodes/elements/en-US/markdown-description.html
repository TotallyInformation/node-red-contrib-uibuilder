Pass-through Markdown from the input data. When sent to the uibuilder node, will be reproduced as HTML in your page(s). Output is wrapped in a div but the optional heading is ignored. Docs.

Markdown will not be rendered as HTML unless the optional Markdown-IT library is loaded.

May be used with the Node-RED core template node.

NOTE: Use with caution, no validity checking is currently done unless the optional DOMPurify library is loaded.
