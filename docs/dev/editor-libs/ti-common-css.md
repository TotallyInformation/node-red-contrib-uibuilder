## Useage

Wrap the whole node config panel in `<div id="ti-edit-panel"></div>`.

## Variables

```css
--uib-node-colour: hsl(248 100% 91%);
--uib-blue: hsl(204 89% 55% / 70%); 
--uib-red: hsl(0 100% 45% / 70%); /* #8f0000; */
```
